# Eureka Dharan School Website

This project is the official website for Eureka Dharan, built with Next.js and TypeScript.

## Highlights
- Modern public-facing pages for about, admission, programs, news, and events
- Admin dashboard for managing content dynamically
- Responsive design for desktop and mobile devices

## Development
Run the app locally with:

```bash
npm install
npm run dev
```

## Deployment
The site is configured for deployment on Vercel and other Node.js hosting platforms.
